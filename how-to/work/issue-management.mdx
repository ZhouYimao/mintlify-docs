## What are Issues?

Issues in Cagen are structured work items that represent:

- **Features** - New functionality to build
- **Bugs** - Problems to fix
- **Tasks** - Work items to complete
- **Improvements** - Enhancements to existing features
- **Research** - Investigation or discovery work

Each issue automatically creates a related discussion channel, keeping work and communication connected.

## Automatic Channel Creation

When you create an issue, Cagen automatically:

### 1. Creates Discussion Channel
```
Issue: "Add dark mode toggle to settings" (#123)
â†“
Channel: "#issue-123-dark-mode-toggle"
```

### 2. Invites Relevant People
- **Assignee** - Person responsible for the work
- **Reporter** - Person who created the issue
- **Stakeholders** - People mentioned in the issue
- **Team Members** - Based on labels and project settings

### 3. Links Issue â†” Channel
- Issue updates appear in channel
- Channel discussions update issue
- Status changes sync bidirectionally

### 4. Sets Up Integrations
- **Memory Bank** - Save important decisions
- **Agent Access** - AI agents can participate
- **Workflow Triggers** - Automated status updates

## API Key Usage & Billing

When AI agents work on issues, API costs are managed through your [payment settings](/how-to/settings/payment-configuration):

### Cost Tracking
```yaml
Per Issue: Track AI usage costs for each issue
Per Agent: Monitor which agents use most resources
Per Team Member: Allocate costs to different users
Per Project: Budget tracking for different initiatives
```

### API Key Priority Order
1. **Project-specific keys** - If issue has dedicated budget
2. **Team shared keys** - Default for team collaboration  
3. **Personal backup keys** - Fallback when team quota exceeded
4. **Organization keys** - Final fallback for enterprise plans

## Troubleshooting

<AccordionGroup>
  <Accordion title="Channel Not Created">
    **Cause:** Issue creation may have failed partway through
    
    **Solution:** Edit issue and save again to trigger channel creation
  </Accordion>
  
  <Accordion title="Missing Notifications">
    **Check:** Notification settings and channel membership
    
    **Solution:** Ensure you're added to issue discussion channel
  </Accordion>
  
  <Accordion title="Agent Not Responding">
    **Check:** API key limits and agent availability
    
    **Solution:** Review billing settings and agent configuration
  </Accordion>
